#! Cropia

Cropia is a Flutter-based mobile application (Android & iOS) that helps smallholder farmers with disease detection, weather alerts, AI advisory (AgriBot), community collaboration, and task management.

## Project structure

- `lib/`
	- `main.dart` - app entry point, route definitions
	- `screens/` - top-level UI screens
		- `home_screen.dart` - main dashboard with bottom navigation and FAB camera
		- `detect_screen.dart` - disease detection (camera / upload)
		- `weather_screen.dart` - forecasts and alerts
		- `agribot_screen.dart` - AI chat assistant placeholder
		- `community_screen.dart` - posts and outbreak reports
		- `tasks_screen.dart` - task reminders and management
		- `profile_screen.dart` - user profile and settings
		- `auth/` - authentication screens
			- `sign_in.dart`
			- `sign_up.dart`

- `android/`, `ios/`, `web/`, `windows/`, `linux/`, `macos/` - platform folders generated by Flutter
- `test/` - widget/unit tests
- `pubspec.yaml` - Dart/Flutter dependencies and assets

## Design

Color scheme: green, black, white
- Active icons: green
- Inactive icons: black
- App uses Material 3 seed color (green)

Icons: Bottom navigation uses filled variants when active and outlined when inactive; active icon is slightly larger and colored green.

## Tools & Frameworks

- Flutter (Dart) - cross-platform mobile SDK
- Recommended editor: VS Code or Android Studio
- Backend (planned): Firebase or Supabase
- Design / prototyping: Figma

## How to run

1. Install Flutter SDK: https://flutter.dev
2. From project root:

```bash
flutter pub get
flutter analyze
flutter run
```
